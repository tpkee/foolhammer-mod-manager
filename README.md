# Tauri + Nuxt Scaffold

A desktop application scaffold using Tauri 2 and Nuxt 4.

## Setup

```bash
pnpm install
```

## Development

Run the Tauri app:
```bash
pnpm dev:tauri
```

Run web only:
```bash
pnpm dev:web
```

## Build

```bash
pnpm build:tauri
```

## Stack

- Tauri 2
- Nuxt 4
- Vue 3
- Tailwind CSS 4
- TypeScript
- i18n
